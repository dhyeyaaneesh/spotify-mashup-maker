# Spotify Mashup Creator 
(1st year - 1st Hackathon)
A Python desktop application that suggests mashup compatible songs using
machine learning and allows basic audio mashup creation.

## Features
- Mashup song recommendation using audio features
- KNN-based similarity search
- Outlier detection for song uniqueness
- Audio playback and mashup generation
- Tkinter GUI

## Tech Stack
- Python
- Tkinter
- Scikit-learn
- Librosa
- Pygame

Dataset not included due to size/licensing.
